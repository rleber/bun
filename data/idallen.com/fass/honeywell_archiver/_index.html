FASS Honeywell Archiver Files 1976-1990

The UW MFCF Honeywell GCOS system had an in-house 9-track Tape Archiver system running until about 1992. You could ask that files be saved into the Archiver. It didn't save your whole account; it only saved the files you asked to be saved.

When the Honeywell was shut down, all the Honeywell tapes, including both the full system save tapes and the Archiver tapes, went to Thinkage Inc. (Peter Fraser, Alan Bowler, Kevin Martin, Keith Dorken, etc.). Around 1992, Alan copied all the Archiver tapes onto a DVD before the tape drive broke. (The dozens and dozens of system save tapes, containing all the accounts as of the date the machine was shut down, are still sitting on the floor of a trailer at Thinkage. They need to be cleaned, run through a 9-track tape drive, then reorganized and collated into actual files.)

This web directory is a quick-and-dirty ASCII dump of just the FASS files found on the Archiver tapes, archived between 1977 and 1990.

Decoding the files

I've only dumped the ASCII Archiver files so far (February 17, 2012). In fact, I've simply dumped *all* the files as if they were ASCII files, which is wrong but does mostly work for most of the ASCII files. I'm not doing a proper dump of the record format yet, and so there are many errors in the dumped files. Many of the text files have duplicate, partial, and deleted records. Use this set of dumps as an idea of what is in the files, but don't take them as the definitive data copy yet. At some point I (or someone else) will do a proper dump program that skips the crap and produces exact ASCII files.

None of the compressed FREEze or HUFFman files are decoded yet. (They are dumped as ASCII, which only works for the names of the items in the FREEze file.) If you want to help decode these files, see the _misc directory for details. You might try writing a program to do the dumping. I did find some B language source for the FREEze program and the HUFFman program that can decode/decompress these file formats. Warning: Even if you convert these B language programs to C, you're still going to have to handle the Honeywell's 36-bit words. The raw tape files are here with an index to the FASS files here. Good luck!

Contact Ian! Allen for more details on any of this stuff.
